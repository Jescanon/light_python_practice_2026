# Индексатор (полный вариант)

Консольная утилита, которая обходит папку и складывает данные в SQLite.

Группа: ПИЭ-41
ФИО: Смараков Александр Владимирович
Вариант: полный

## Что где лежит
- main.py - запуск, разбор аргументов
- config.py - пути к базе и логам
- database/db.py - sqlite, создание таблиц
- logs_service/logs.py - логи
- service/ - тут будет сканер

База data/app.db и лог logs/indexer.log создаются сами при запуске.

## Запуск

Запускать из корня проекта, обязательно через -m (просто python src/main.py не работает, ругается на импорты):

```
python -m src.main scan <папка>
```

Флаги -v и --db идут ДО слова scan, а не после (сам на это напоролся):

```
python -m src.main -v scan C:\папка
python -m src.main --db C:\tmp\my.db scan C:\папка
```

Фильтры у scan:
- --ext .py,.txt - по расширению
- --name отчет - по куску имени

-v это подробный лог, --db можно не писать, тогда база в data/app.db.
